## Summary

Defines the `ReceiptList` struct and its methods for managing an ordered collection of receipts with a Merkle root for integrity verification.

## Specification

### Struct

```rust
pub struct ReceiptList {
    pub receipts: Vec<Receipt>,
    pub root: Bytes32,
}
```

### new

```rust
impl ReceiptList {
    pub fn new() -> Self {
        Self {
            receipts: Vec::new(),
            root: EMPTY_ROOT,
        }
    }
}
```

Creates an empty receipt list with `root` set to `EMPTY_ROOT`.

### from_receipts

```rust
impl ReceiptList {
    pub fn from_receipts(receipts: Vec<Receipt>) -> Self {
        let root = compute_receipts_root(&receipts);
        Self { receipts, root }
    }
}
```

Creates a receipt list from a vector of receipts and computes the Merkle root.

### push

```rust
impl ReceiptList {
    pub fn push(&mut self, receipt: Receipt) {
        self.receipts.push(receipt);
    }
}
```

Appends a receipt to the list. Note: does not recompute the root; call `finalize()` when done.

### finalize

```rust
impl ReceiptList {
    pub fn finalize(&mut self) {
        self.root = compute_receipts_root(&self.receipts);
    }
}
```

Recomputes the Merkle root over the current receipts.

### get

```rust
impl ReceiptList {
    pub fn get(&self, index: usize) -> Option<&Receipt> {
        self.receipts.get(index)
    }
}
```

Returns the receipt at the given index, or `None` if out of bounds.

### get_by_tx_id

```rust
impl ReceiptList {
    pub fn get_by_tx_id(&self, tx_id: Bytes32) -> Option<&Receipt> {
        self.receipts.iter().find(|r| r.tx_id == tx_id)
    }
}
```

Returns the first receipt matching the given transaction ID, or `None` if not found.

## Acceptance Criteria

- [ ] `ReceiptList` contains `receipts: Vec<Receipt>` and `root: Bytes32`.
- [ ] `new()` creates an empty list with `EMPTY_ROOT`.
- [ ] `from_receipts(vec)` stores receipts and computes the correct Merkle root.
- [ ] `push(receipt)` appends the receipt to the internal vector.
- [ ] `finalize()` recomputes the root to reflect current receipts.
- [ ] After `push` + `finalize`, root matches `from_receipts` with the same receipts.
- [ ] `get(index)` returns `Some(&Receipt)` for valid indices.
- [ ] `get(index)` returns `None` for out-of-bounds indices.
- [ ] `get_by_tx_id(tx_id)` returns `Some(&Receipt)` when a matching receipt exists.
- [ ] `get_by_tx_id(tx_id)` returns `None` when no receipt matches.

## Implementation Notes

- `EMPTY_ROOT` is a constant representing the Merkle root of an empty list (typically all zeros or a defined hash constant).
- `push` intentionally does not recompute the root for performance; the caller must call `finalize()` when the list is complete.
- `from_receipts` is a convenience method that combines construction and finalization.
- `get_by_tx_id` uses linear search; this is acceptable for typical block sizes.
