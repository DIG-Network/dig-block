## Summary

`L2BlockHeader::genesis()` constructs the genesis block header with deterministic initial values. The genesis block has no parent (uses network_id as parent_hash), empty state roots, zero costs/counts, and the current wall-clock timestamp.

## Specification

```rust
use std::time::{SystemTime, UNIX_EPOCH};

const EMPTY_ROOT: Bytes32 = /* SHA256 of empty Merkle tree */;
const ZERO_HASH: Bytes32 = Bytes32::default(); // all zeros

impl L2BlockHeader {
    pub fn genesis(
        network_id: Bytes32,
        l1_height: u32,
        l1_hash: Bytes32,
    ) -> Self {
        let timestamp = SystemTime::now()
            .duration_since(UNIX_EPOCH)
            .expect("system clock before epoch")
            .as_secs();

        Self {
            version: Self::detect_version(),
            height: 0,
            epoch: 0,
            parent_hash: network_id,
            state_root: EMPTY_ROOT,
            transactions_root: EMPTY_ROOT,
            receipts_root: EMPTY_ROOT,
            filter_hash: EMPTY_ROOT,
            extension_data: ZERO_HASH,
            transaction_count: 0,
            total_cost: 0,
            total_fees: 0,
            l1_inclusion_proof: None,
            l1_checkpoint_proof: None,
            timestamp,
            l1_height,
            l1_hash,
            network_id,
        }
    }

    fn detect_version() -> u32 {
        // Auto-detect from crate version or protocol constant
        env!("CARGO_PKG_VERSION_MAJOR").parse().unwrap_or(1)
    }
}
```

- [ ] `epoch` is set to 0
- [ ] `parent_hash` is set to `network_id`
- [ ] `state_root` is set to `EMPTY_ROOT`
- [ ] `transactions_root` is set to `EMPTY_ROOT`

- [ ] `filter_hash` is set to `EMPTY_ROOT`
- [ ] `extension_data` is set to `ZERO_HASH`
- [ ] All counts and costs are set to 0
- [ ] All L1 proofs are set to `None`
- [ ] `timestamp` is set to current wall-clock time
- [ ] `version` is auto-detected from crate version

## Implementation Notes

- `EMPTY_ROOT` is the SHA-256 hash of the empty Merkle tree. This is a well-known constant and should be defined once in the crate.
- The genesis block is special: it has no parent block, so `parent_hash` is set to the `network_id` to establish the chain identity. This prevents cross-network block replay.
- `timestamp` uses wall-clock time, which means genesis blocks created at different times will have different hashes.
