## Summary

Defines the query methods on `AttestedBlock` for checking signing progress, soft finality, and block identity via hash delegation.

## Specification

### signing_percentage

```rust
impl AttestedBlock {
    pub fn signing_percentage(&self) -> u64 {
        self.signer_bitmap.signing_percentage()
    }
}
```

Returns an integer in the range 0-100 representing the percentage of validators that have signed.

### has_soft_finality

```rust
impl AttestedBlock {
    pub fn has_soft_finality(&self, threshold_pct: u64) -> bool {
        self.signer_bitmap.has_threshold(threshold_pct)
    }
}
```

Returns `true` if the signing percentage meets or exceeds `threshold_pct`.

### hash

```rust
impl AttestedBlock {
    pub fn hash(&self) -> Bytes32 {
        self.block.hash()
    }
}
```

Delegates to `block.hash()` so the attested block is identified by its underlying L2 block hash.

## Acceptance Criteria

- [ ] `signing_percentage()` returns a value in range 0-100.
- [ ] `signing_percentage()` delegates to `signer_bitmap.signing_percentage()`.
- [ ] `has_soft_finality(threshold_pct)` returns `true` when signing percentage >= threshold.
- [ ] `has_soft_finality(threshold_pct)` returns `false` when signing percentage < threshold.
- [ ] `hash()` returns the same value as `block.hash()`.

## Implementation Notes

- `signing_percentage` and `has_soft_finality` are convenience wrappers that delegate to `SignerBitmap` methods.
- The hash delegation ensures that block identity is consistent whether accessed through `L2Block` or `AttestedBlock`.

## Test Plan

| Test Case                         | Description                                              | Expected Result       |
|-----------------------------------|----------------------------------------------------------|-----------------------|
| Percentage with no signers        | Call `signing_percentage()` on fresh AttestedBlock       | Returns 0             |
| Percentage with all signers       | Set all validators as signed, check percentage           | Returns 100           |
| Percentage with partial signers   | Set half validators as signed, check percentage          | Returns ~50           |
| Soft finality below threshold     | Check `has_soft_finality(67)` with 50% signing           | Returns `false`       |
| Soft finality at threshold        | Check `has_soft_finality(67)` with 67% signing           | Returns `true`        |
| Soft finality above threshold     | Check `has_soft_finality(67)` with 80% signing           | Returns `true`        |
| Hash delegation                   | Compare `attested_block.hash()` with `block.hash()`     | Values are equal      |
