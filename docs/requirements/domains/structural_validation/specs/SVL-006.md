## Summary

`L2Block::validate_structure()` must perform ten integrity sub-checks covering Merkle root recomputation, duplicate detection, slash proposal limits, filter hash verification, and overall size enforcement.

## Specification

The following checks are performed in order:

```rust
// Step 3: Recompute spends_root
let computed_spends_root = MerkleTree::new(&spend_hashes).root();
if self.header.spends_root != computed_spends_root {
    return Err(BlockError::InvalidSpendsRoot);
}

// Step 6: No duplicate outputs (Chia Check 13)
let mut seen_outputs = HashSet::new();
for coin in &all_additions {
    if !seen_outputs.insert(coin.coin_id()) {
        return Err(BlockError::DuplicateOutput(coin.coin_id()));
    }
}

// Step 7: No double spends (Chia Check 14)
let mut seen_spends = HashSet::new();
for removal in &all_removals {
    if !seen_spends.insert(*removal) {
        return Err(BlockError::DoubleSpendInBlock(*removal));
    }
}

// Step 8: Recompute additions_root
let computed_additions_root = compute_merkle_set_root(&additions_by_puzzle_hash);
if self.header.additions_root != computed_additions_root {
    return Err(BlockError::InvalidAdditionsRoot);
}

// Step 9: Recompute removals_root
let computed_removals_root = compute_merkle_set_root(&all_removals);
if self.header.removals_root != computed_removals_root {
    return Err(BlockError::InvalidRemovalsRoot);
}

// Step 10: Recompute filter_hash (BIP158)
let computed_filter = build_gcs_filter(&all_additions, &all_removals);
if self.header.filter_hash != computed_filter.hash() {
    return Err(BlockError::InvalidFilterHash);
}

// Step 11: Slash proposal count limit
if self.slash_proposal_payloads.len() > MAX_SLASH_PROPOSALS_PER_BLOCK {
    return Err(BlockError::TooManySlashProposals);
}

// Step 12: Individual slash proposal payload size
for payload in &self.slash_proposal_payloads {
    if payload.len() > MAX_SLASH_PROPOSAL_PAYLOAD_BYTES {
        return Err(BlockError::SlashProposalPayloadTooLarge);
    }
}

// Step 14: Recompute slash_proposals_root
let computed_slash_root = compute_slash_proposals_root(&self.slash_proposal_payloads);
if self.header.slash_proposals_root != computed_slash_root {
    return Err(BlockError::InvalidSlashProposalsRoot);
}

// Step 15: Serialized size check
let serialized_size = self.serialized_size();
if serialized_size > MAX_BLOCK_SIZE {
    return Err(BlockError::TooLarge {
        size: serialized_size,
        max: MAX_BLOCK_SIZE,
    });
}
```

## Acceptance Criteria

- [ ] `validate_structure()` returns `Err(InvalidSpendsRoot)` when spends_root does not match recomputed value.
- [ ] `validate_structure()` returns `Err(DuplicateOutput)` when two additions share the same coin ID.
- [ ] `validate_structure()` returns `Err(DoubleSpendInBlock)` when the same coin is spent twice.
- [ ] `validate_structure()` returns `Err(InvalidAdditionsRoot)` when additions_root does not match.
- [ ] `validate_structure()` returns `Err(InvalidRemovalsRoot)` when removals_root does not match.
- [ ] `validate_structure()` returns `Err(InvalidFilterHash)` when filter_hash does not match BIP158 recomputation.
- [ ] `validate_structure()` returns `Err(TooManySlashProposals)` when count exceeds `MAX_SLASH_PROPOSALS_PER_BLOCK`.
- [ ] `validate_structure()` returns `Err(SlashProposalPayloadTooLarge)` when any payload exceeds `MAX_SLASH_PROPOSAL_PAYLOAD_BYTES`.
- [ ] `validate_structure()` returns `Err(InvalidSlashProposalsRoot)` when slash_proposals_root does not match.
- [ ] `validate_structure()` returns `Err(TooLarge)` when serialized size exceeds `MAX_BLOCK_SIZE`.
- [ ] `validate_structure()` returns `Ok` for a well-formed block passing all sub-checks.

## Implementation Notes

- The additions_root uses `compute_merkle_set_root` with coins grouped by `puzzle_hash`, following Chia's merkle set construction.
- The filter_hash uses BIP158 Golomb-Coded Set (GCS) construction over additions and removals.
- Duplicate output detection corresponds to Chia Check 13; double spend detection to Chia Check 14.
- The final size check (Step 15) re-verifies the header's `block_size` field against actual serialized length.
