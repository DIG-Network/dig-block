## Summary

`L2Block::validate_structure()` must verify that the four count fields in the block header exactly match the actual counts computed from the block body.

## Specification

Four count fields are validated:

```rust
// Step 2: Spend bundle count
if self.header.spend_bundle_count as usize != self.spend_bundles.len() {
    return Err(BlockError::SpendBundleCountMismatch {
        header: self.header.spend_bundle_count,
        actual: self.spend_bundles.len() as u32,
    });
}

// Step 4: Additions count
let computed_additions: usize = self.spend_bundles.iter()
    .flat_map(|sb| &sb.coin_spends)
    .map(|cs| cs.additions().len())
    .sum();
if self.header.additions_count as usize != computed_additions {
    return Err(BlockError::AdditionsCountMismatch {
        header: self.header.additions_count,
        actual: computed_additions as u32,
    });
}

// Step 5: Removals count
let computed_removals: usize = self.spend_bundles.iter()
    .map(|sb| sb.coin_spends.len())
    .sum();
if self.header.removals_count as usize != computed_removals {
    return Err(BlockError::RemovalsCountMismatch {
        header: self.header.removals_count,
        actual: computed_removals as u32,
    });
}

// Step 13: Slash proposal count
if self.header.slash_proposal_count as usize != self.slash_proposal_payloads.len() {
    return Err(BlockError::SlashProposalCountMismatch {
        header: self.header.slash_proposal_count,
        actual: self.slash_proposal_payloads.len() as u32,
    });
}
```

Each mismatch produces a distinct error variant carrying both the header value and the computed value.

## Acceptance Criteria

- [ ] `validate_structure()` returns `Ok` when all four counts match.
- [ ] `validate_structure()` returns `Err(SpendBundleCountMismatch)` when `spend_bundle_count` is wrong.
- [ ] `validate_structure()` returns `Err(AdditionsCountMismatch)` when `additions_count` is wrong.
- [ ] `validate_structure()` returns `Err(RemovalsCountMismatch)` when `removals_count` is wrong.
- [ ] `validate_structure()` returns `Err(SlashProposalCountMismatch)` when `slash_proposal_count` is wrong.
- [ ] Each error carries both the header-declared count and the computed count.

## Implementation Notes

- Additions are computed by flattening all coin spends across all spend bundles and summing their output coins.
- Removals are computed by counting the coin spends themselves (each spend removes one coin).
- These checks run early in `validate_structure()` to fail fast before expensive Merkle computations.
