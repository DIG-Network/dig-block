# STV-005: Height/Time Lock Evaluation

- **Normative:** [NORMATIVE.md](../NORMATIVE.md#stv-005-heighttime-lock-evaluation)
- **Verification:** [VERIFICATION.md](../VERIFICATION.md)
- **Tracking:** [TRACKING.yaml](../TRACKING.yaml)
- **Spec:** SPEC Section 7.5.4

## Summary

Pending assertions from Tier 2 (height and time locks) MUST be evaluated against the current chain context. Each assertion type has specific comparison semantics involving chain height, chain timestamp, and per-coin confirmed height/timestamp.

## Specification

```rust
fn evaluate_pending_assertions(
    &self,
    exec: &ExecutionResult,
    coins: &dyn CoinLookup,
) -> Result<(), BlockError> {
    let chain_height = self.header.height;
    let chain_timestamp = self.header.timestamp;

    for assertion in &exec.pending_assertions {
        let coin_state = assertion.coin_id
            .and_then(|id| coins.get_coin_state(&id));

        match assertion.kind {
            // Absolute height assertions
            AssertionKind::HeightAbsolute(h) => {
                if !(chain_height >= h) {
                    return Err(BlockError::AssertionFailed {
                        kind: "ASSERT_HEIGHT_ABSOLUTE".into(),
                        expected: h.to_string(),
                        actual: chain_height.to_string(),
                    });
                }
            }
            // Relative height assertions
            AssertionKind::HeightRelative(h) => {
                let confirmed = coin_state
                    .ok_or(BlockError::CoinNotFound { coin_id: assertion.coin_id.unwrap() })?
                    .created_height;
                if !(chain_height >= confirmed + h) {
                    return Err(BlockError::AssertionFailed {
                        kind: "ASSERT_HEIGHT_RELATIVE".into(),
                        expected: format!(">= {}", confirmed + h),
                        actual: chain_height.to_string(),
                    });
                }
            }
            // Absolute time assertions
            AssertionKind::SecondsAbsolute(t) => {
                if !(chain_timestamp >= t) {
                    return Err(BlockError::AssertionFailed {
                        kind: "ASSERT_SECONDS_ABSOLUTE".into(),
                        expected: t.to_string(),
                        actual: chain_timestamp.to_string(),
                    });
                }
            }
            // Relative time assertions
            AssertionKind::SecondsRelative(t) => {
                let coin_ts = coin_state
                    .ok_or(BlockError::CoinNotFound { coin_id: assertion.coin_id.unwrap() })?
                    .timestamp;
                if !(chain_timestamp >= coin_ts + t) {
                    return Err(BlockError::AssertionFailed {
                        kind: "ASSERT_SECONDS_RELATIVE".into(),
                        expected: format!(">= {}", coin_ts + t),
                        actual: chain_timestamp.to_string(),
                    });
                }
            }
            // Before absolute height
            AssertionKind::BeforeHeightAbsolute(h) => {
                if !(chain_height < h) {
                    return Err(BlockError::AssertionFailed {
                        kind: "BEFORE_HEIGHT_ABSOLUTE".into(),
                        expected: format!("< {}", h),
                        actual: chain_height.to_string(),
                    });
                }
            }
            // Before relative height
            AssertionKind::BeforeHeightRelative(h) => {
                let confirmed = coin_state
                    .ok_or(BlockError::CoinNotFound { coin_id: assertion.coin_id.unwrap() })?
                    .created_height;
                if !(chain_height < confirmed + h) {
                    return Err(BlockError::AssertionFailed {
                        kind: "BEFORE_HEIGHT_RELATIVE".into(),
                        expected: format!("< {}", confirmed + h),
                        actual: chain_height.to_string(),
                    });
                }
            }
            // Before absolute time
            AssertionKind::BeforeSecondsAbsolute(t) => {
                if !(chain_timestamp < t) {
                    return Err(BlockError::AssertionFailed {
                        kind: "BEFORE_SECONDS_ABSOLUTE".into(),
                        expected: format!("< {}", t),
                        actual: chain_timestamp.to_string(),
                    });
                }
            }
            // Before relative time
            AssertionKind::BeforeSecondsRelative(t) => {
                let coin_ts = coin_state
                    .ok_or(BlockError::CoinNotFound { coin_id: assertion.coin_id.unwrap() })?
                    .timestamp;
                if !(chain_timestamp < coin_ts + t) {
                    return Err(BlockError::AssertionFailed {
                        kind: "BEFORE_SECONDS_RELATIVE".into(),
                        expected: format!("< {}", coin_ts + t),
                        actual: chain_timestamp.to_string(),
                    });
                }
            }
        }
    }

    Ok(())
}
```

## Acceptance Criteria

- [ ] `ASSERT_HEIGHT_ABSOLUTE(h)`: passes when `chain_height >= h`
- [ ] `ASSERT_HEIGHT_RELATIVE(h)`: passes when `chain_height >= confirmed_height + h`
- [ ] `ASSERT_SECONDS_ABSOLUTE(t)`: passes when `chain_timestamp >= t`
- [ ] `ASSERT_SECONDS_RELATIVE(t)`: passes when `chain_timestamp >= coin_timestamp + t`
- [ ] `BEFORE_HEIGHT_ABSOLUTE(h)`: passes when `chain_height < h`
- [ ] `BEFORE_HEIGHT_RELATIVE(h)`: passes when `chain_height < confirmed_height + h`
- [ ] `BEFORE_SECONDS_ABSOLUTE(t)`: passes when `chain_timestamp < t`
- [ ] `BEFORE_SECONDS_RELATIVE(t)`: passes when `chain_timestamp < coin_timestamp + t`
- [ ] Failed assertions reject with `BlockError::AssertionFailed`
- [ ] Relative assertions require coin state lookup for confirmed_height/timestamp
- [ ] Chia parity with Check 21

## Implementation Notes

- Relative assertions require looking up the coin's creation height and timestamp from the coin state database. For ephemeral coins (created in the same block), the confirmed height is the current block height.
- The BEFORE_* assertions create time windows: a coin can only be spent before a certain height/time. Combined with ASSERT_* assertions, they create bounded validity windows.
